# Laravel AdminLTE Starter
[Admin LTE] (https://almsaeedstudio.com/preview) Starter for [Laravel] (https://laravel.com/)

## Installation

``` bash
# install composer
composer install

# install npm
npm install

# install bower
bower install

# run gulp for compile asset
gulp

# migrate users table
php artisan migrate

# seed user
php artisan db:seed

# run
php artisan serve
```
